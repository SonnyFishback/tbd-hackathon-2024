# Application

## Development Environment:

1. Install Prerequisites:

- Node Version Manager.
- Node.js
- PNPM

2. Gather required credentials/access:

- Supabase
- Vercel
- Github

3. Clone project

4. Install dependencies

```bash
    cd ./application
    pnpm install
```

5. Run development server:

```bash
    pnpm dev
```
